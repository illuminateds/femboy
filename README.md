# FEMBOY — Anti-Fraud Portfolio

Готовый адаптивный одностраничный сайт для запуска в Visual Studio Code.

## Запуск
1. Откройте папку проекта в VS Code.
2. Установите расширение **Live Server**.
3. Нажмите правой кнопкой по `index.html`.
4. Выберите **Open with Live Server**.

## Структура
- `index.html` — разметка и содержимое прайс-листа.
- `css/style.css` — дизайн, адаптивность и анимации.
- `js/script.js` — мобильное меню, появление блоков и модальное окно.
- `assets/avatar.jpg` — аватар профиля.

## Производительность
- Нет тяжёлых библиотек и внешних шрифтов.
- Анимации используют преимущественно `transform` и `opacity`.
- IntersectionObserver отключает обработку элементов после появления.
- На мобильных отключены тяжёлые декоративные эффекты.
- Поддерживаются `prefers-reduced-motion` и `prefers-reduced-data`.
